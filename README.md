# BE-Monitor

前端埋点监控后台，提供可视化数据报表，报错趋势图，以及报错聚合性分析报表。

## 设计目的

1.方便更多的前端开发快速接入前端埋点。

2.为更多中小型的项目增加前端的报错监控。

3.作为一款开源产品帮助更多的项目提升稳定性。

## 使用

在前端前端项目中接入 SDK ，可以 [参考 FE-Monitor](https://github.com/MriLiuJY/monitorFE) 

之后配置 config 中的 url 到该后台的服务器域名上即可。
